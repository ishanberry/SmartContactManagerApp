About the Smart Contact Manager

The Smart Contact Manager is a comprehensive application designed to streamline the management of contacts for individuals and organizations. This project aims to provide a secure, efficient, and user-friendly interface for handling contact information, complete with advanced features and integrations.

Purpose:

The primary purpose of the Smart Contact Manager is to simplify the process of managing contact information. Whether you are an individual looking to organize personal contacts or a business needing to maintain client information, this application offers a reliable solution.

Key Features:

  • User Authentication via OAuth2 (Google and GitHub) and Database
  • Full CRUD Operations for Contacts
  • Secure Password Encryption and Validation
  • Contact Image Storage with Cloudinary Integration
  • AJAX-powered Modal Views for Enhanced User Experience
  • Separate Admin Portal for User Management
  • Email Verification for New User Registrations
  • Search and Pagination for Efficient Data Navigation
  • Export Contacts to Excel for Easy Data Handling

Upcoming Features:

  • Voice-based Contact Management Using AI-Powered Speech Recognition
  • Advanced Analytics Dashboard for Contact Insights

Technologies Used:

  • Java and Spring Boot for Backend Development
  • Thymeleaf for Server-Side Templating
  • Tailwind CSS for Modern, Responsive UI
  • MySQL for Database Management
  • Heroku for Cloud Deployment
